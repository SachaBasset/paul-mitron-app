# 🥐 Paul Mitron - Une histoire interactive pour les enfants

Paul Mitron est une histoire illustrée et interactive où le jeune lecteur effectue des choix pour avancer dans l'aventure.
L'application a été développée avec le moteur de jeu Ink et vibe-codé en React native.
Les illustrations ont été produite par IA.
